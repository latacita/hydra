Una vez que se ha implementado la aplicación completamente, el último paso para finalizar este proyecto es exportarla para que pueda ser utilizada en otros computadores.

Para llevar a cabo esta tarea se ha optado por la creación de un \emph{update site}, que permite instalar la aplicación desde la opción \emph{Install New Software} del propio menú de Eclipse. Consideramos que esta era la opción más cómoda de distribuir nuestra aplicación. El archivo \emph{updateSite.zip} del CD adjunto contiene los ficheros necesarios para llevar a cabo la instalación de nuestro editor.

Además, hemos creado un pequeño manual de usuario que explica de manera rápida y con muchas imágenes como empezar a utilizar nuestra aplicación. También hemos grabado un videotutorial que muestra la aplicación en funcionamiento. Tanto el manual como el vídeo están disponibles en el CD adjunto, bajo el nombre de \emph{manual.pdf} y \emph{video.mp4}.
